<h2 align="center">🎯 Deep Work</h2>

<div align="center">
  <a href="https://betting-ai-pro.vercel.app">
    <img src="https://betting-ai-pro.vercel.app/BettingAi.png" width="160" alt="BettingAi" />
  </a>
  <h3><a href="https://betting-ai-pro.vercel.app">BettingAIPro</a> &nbsp;<sub>· <em>live</em></sub></h3>
  <p><em>An AI sports prediction app — a two-model pipeline that turns live football data into probability-graded bets with Kelly-sized stakes.</em></p>
</div>

<br/>

<p align="center">
  <img src="./pipeline.svg" alt="BettingAIPro prediction pipeline" width="85%" />
</p>

<br/>

> **The Scout** never predicts. It reads intel *(injuries, suspensions, form, rotation risk)* and emits bounded numeric deltas — max `±2.5 xG` on injuries, `±1.0` on suspensions — into a deterministic **Dixon-Coles Poisson engine**. 
>
> **The Judge** then re-grades the math. 

<br/>

<p align="center">
  <code>Extractive LLM</code> &nbsp;➔&nbsp; <code>Pure Math</code> &nbsp;➔&nbsp; <code>Judgment LLM</code>
</p>

<p align="center">
  <em>Every prediction is auditable down to which intel line moved which xG component.</em>
</p>
